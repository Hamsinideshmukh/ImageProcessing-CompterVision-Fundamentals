# 1) SomeCVAlgorithms.ipynb

- Implemented and compared noise reduction techniques including Mean, Median, Gaussian and Non-Local Means filtering using PSNR and SSIM metrics.  
- Performed edge detection using Prewitt, Sobel, Canny and Marr–Hildreth methods and applied edge-based image sharpening.  
- Simulated Gaussian and Salt & Pepper noise and evaluated denoising performance quantitatively.  
- Implemented image segmentation techniques such as GraphCut and Mean Shift for region-based object extraction.  
- Applied feature detection and matching using SIFT, ORB and AKAZE for keypoint extraction and image matching tasks.
